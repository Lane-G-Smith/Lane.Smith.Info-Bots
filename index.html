DISCORD BOTS
Developed By Lane Smith

As a new & learning developer, one of the initial tasks I undertook was to create automated bots for Discord, a platform that I utilize on a daily basis. Many of these bots utilize the Chat-GPT REST API from OpenAI. The Chat-GPT API allows the desired trigger & response format as well as personality logic to be built right into the POST request. Other bots leverage existing REST APIs that return data (random quotes in this instance) from a database when triggered. At present, my focus is on developing my own REST APIs as well as formatting my own databases. Managing my own APIs will enable me to enhance the functionality of my bots in the future.

Code-Buddy: A Virtual Code Interview

I made Code-Buddy to help me get ready for coding job interviews. This Bot will give random questions and code challenges based on what you want to practice and how difficult you want it to be. After you try to solve the challenge, Code-Buddy gives you the answer and explains how to solve it. Code-Buddy even provides links to documentation where you can go to learn more. To use Code-Buddy, just tell it what subject you want to practice, like "React" or "Javascript," and how hard you want the question to be, like "1/10" or "7/10." Then Code-Buddy will give you a random question or code challenge. If you want the answer and a detailed explanation, you just type "answer" after the question is given.

GPT-Buddy

I made GPT-Buddy so that I could have Chat-GPT at my fingertips at all times, even on my phone! By creating a dedicated server just for me and the Bot as well as dedicated text channels for each subject that I am interested in I can look back on all of my previous questions and answers with ease. I also created a dedicated channel for the Bot to post links to resources that I might find helpful. This Bot was programmed to be a personal assistant and will answer all questions as well as explain the answers in detail. When asked, GPT-Buddy will also provide links to sources and documentation when needed. Simply start each query with "gpt-buddy" (not case sensitive) followed by your question. GPT-Buddy started with GPT-3.5, then was upgraded to Turbo, and is now currently running GPT-3.5-Turbo-0613. When I get access to the GPT-4 REST API I will upgrade again.
